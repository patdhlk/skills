# Skills

Flat layout: one directory per skill, each with a `SKILL.md`
(`skills/<name>/SKILL.md`) — consumed by both the Claude Code plugin
manifest and `npx skills add patdhlk/skills` (ADR_0013).

No skills are implemented yet. The v1 backlog is tracked as `.. issue::`
directives in [`../spec/issues/index.rst`](../spec/issues/index.rst) —
all currently `ready-for-agent`.
